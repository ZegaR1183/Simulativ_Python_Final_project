<h2 id="base_python">Итоговый проект: Базовый Python</h2>
Даный проект содержит скрипт, в котором происходит:

- Сбор данных с помощью API,
- Обработка и подготовка данных к загрузке в базу данных.
- Обработанные данные загружаються в локальную базу PostgreSQL.
- Во время обработки сохраняться лог работы скрипта с отлавливанием ошибок и выводом промежуточных стадий.
- В Google Sheets загружаются информационные агрегированные данные
- Настроено оповещение на почту  

Используюемые библиотеки Python:  
requests, json, logging, gspread, smtplib, ssl, psycopg2, tenacity,oauth2client, email.


<h2 id="python_pandas">Итоговый проект: Python для анализа данных</h2>
В данном проекте данные загружаются из исходных файлов Excel.  

Формируетя датафрейм по каждому файлу.  
Объединение 2-х датафреймов в один, по общему столбцу.  

Описание функций в данном скрипте:
 - ***popular_product_group*** - Функция определения самой ходовой группы товара и отображение на графике;
 - ***sales_by_subcategori*** - Распределение продаж по подкатегориям и отображение на графике;
 - ***average_check*** - Средний чек в заданную дату
 - ***promo_share_category*** - Доля промо в заданной категории с отображением на графике;
 - ***calculate_margin_by_category*** - Расчёт маржи по категориям с построение барчарта
 - ***abc_analysis*** - ABC анализ

Используюемые библиотеки Python:  
pandas, matplotlib.
